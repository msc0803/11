# Windows 打包说明

## 第一步：安装 Python

1. 下载 Python 3.10：https://www.python.org/downloads/release/python-31011/
   - 选 `Windows installer (64-bit)` 下载
2. 安装时 **必须勾选** `Add Python to PATH`，然后点 Install Now

## 第二步：拷贝文件

把以下文件拷贝到 Windows 电脑同一个文件夹里：
- `gui_app.py`
- `api_core.py`
- `build_win.bat`

## 第三步：运行打包脚本

右键 `build_win.bat` → **以管理员身份运行**

等待约 3-5 分钟，自动完成：
- 安装所有依赖（PyQt6、pandas、requests 等）
- 打包成单个 exe 文件

## 第四步：找到 exe

打包完成后，exe 在同目录的 `dist\` 文件夹里：`视频查询工具.exe`

直接双击运行即可，无需安装任何东西。

---

## 注意事项

- 使用时需要打开 Chrome 并登录 sucaiwang.zhishangsoft.com（用于读取 Token）
- Chrome 需要以调试模式启动，在命令行运行：
  ```
  "C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222
  ```
  或者创建一个快捷方式，在目标后面加上 `--remote-debugging-port=9222`
